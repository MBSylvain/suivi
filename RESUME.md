# Reprendre le projet "SuiviJob"

Ce fichier résume l'état du projet et les commandes utiles pour reprendre le travail plus tard.

## État Actuel (19/01/2026)
Le projet est **terminé** et prêt pour la mise en ligne.
- **Fonctionnalités** : CRUD complet, Recherche, Filtres, Upload de documents, Authentification.
- **Sécurité** : Base de données sécurisée (RLS), accès protégé par login.

## Commandes Utiles

### 1. Relancer le projet en local
Ouvrez le terminal dans le dossier du projet (`c:\Users\sylva\suivi`) et lancez :
```bash
npm run dev
```
L'application sera accessible sur `http://localhost:5173`.

### 2. Déploiement
Un guide complet pour déployer sur **Vercel** est disponible sur le disque (voir `deployment_guide.md` dans mes souvenirs).
